## Hakkında

Vidius Project; web ve mobil uygulamalar için bağımsız, dış kaynaklı yazılım test danışmanlığı sunar.

17 yıllık QA deneyimiyle manuel ve otomasyon testlerini birleştirerek yazılımınızdaki riskleri erken aşamada tespit eder, raporlar ve çözüm önerileri sunarız.

## Uzmanlık Alanları

 ✅ Manuel Test & Keşifsel Test
✅ Web Otomasyon Testi — Playwright / TypeScript
✅ Android Mobil Otomasyon — Maestro
 ✅ Test Senaryosu & Bug Raporu Hazırlama
 ✅ Test Stratejisi & Süreç Danışmanlığı
